MSAP Multimedia Project

Audio-Reactive Image Glitch Lab

Interaktívna webová aplikácia, ktorá prepája spracovanie obrazu a analýzu zvuku. Obrázok sa deformuje podľa hlasitosti a frekvenčného spektra nahratého audio súboru.

1. Image processing

Obrázok sa načíta do HTML Canvas elementu, kde sa dá ďalej upravovať po jednotlivých riadkoch a pixeloch.

2. Audio analysis

Web Audio API analyzuje audio signál a získava hodnoty hlasitosti a frekvenčného spektra.

3. Reactive glitch

Získané audio dáta riadia vizuálne efekty ako RGB posun, vlnové skreslenie, zmena jasu a glitch deformácia.
